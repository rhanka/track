# SELF-AUDIT GATE — run before presenting any full dossier

Loop: run the checklist → if any item fails, FIX the dossier → re-run. Do not present until every box is checked.
If an item cannot be honestly completed, label the dossier **Incomplete** and ask for the missing fact/criterion — never ask for validation of an incomplete dossier.

## Checklist

- [ ] **FACT / JUDGMENT tags** — every load-bearing claim is tagged `[FACT]` (verifiable, source cited) or `[JUDGMENT]` (my read). No untagged assertions.
- [ ] **Count-symmetry on pros** — each option (incl. rejected) has roughly as many genuine "case FOR" points as the others. I have not starved alternatives to flatter my pick.
- [ ] **Strongest case AGAINST my recommendation** — present, non-empty, and the *best* version (not a strawman).
- [ ] **What would overturn it** — what would have to be true for the owner to reject my recommendation.
- [ ] **Pre-mortem** — one paragraph: "six months later this failed because…".
- [ ] **Agent-interest disclosure** — I named what is easiest / faster / less risky *for me, the presenter*, and flagged any place the recommendation benefits me more than the owner.
- [ ] **Owner interest stated separately** — value · integrity · risk · future optionality · cost.
- [ ] **Attendus covered** — every owner criterion has a `covered by`; no open `gap` (or the gap is the explicit decision asked).
- [ ] **Double-instruction** — for a complex design decision, independent Codex + Opus passes were obtained and synthesized (disagreements surfaced, not blended away). If absent on a hard-trigger call: dossier marked **Incomplete**.
- [ ] **No fabricated evidence** — no faked `comprehension[]`; no bridge principal recorded as the decider.

## Red flags — STOP and rebuild the dossier
- The "case AGAINST" reads weaker than the "case FOR" for the same option.
- One option has visibly fewer FOR points and I can't justify why.
- A claim I'm leaning on has no source and isn't tagged `[JUDGMENT]`.
- The recommendation happens to be the cheapest path *for me* and I haven't said so.
- I'm about to present a chip/option menu with no prose, stakes, or attendus.
